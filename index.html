Portfolio

  • Next.js + AI

    Agentic VR Form App

    AI / Next.js / TypeScript

    A Progressive Web Application developed for the Sight Center of Northwest Ohio that allows blind rehabilitation providers to describe sessions in natural language and have structured documentation populated automatically. Built with Next.js and the Vercel AI SDK, the application uses a two-stage LLM pipeline: one call performs schema-driven field extraction with confidence scoring, and a second generates a conversational follow-up question guiding the provider through any remaining required fields. Providers can also switch to a standard manual entry form at any time, with both modes sharing the same underlying state.

  • PHP + MySQL

    Field Inventory Tracker

    Web App / PHP / MySQL

    A mobile-first web application built for field workers at an industrial manufacturing facility to log physical inventory counts against SAP reference data from their phones and tablets. Built in PHP and MySQL with vanilla JavaScript, the application includes local authentication, role-based access control, concurrent-safe storage, peer visibility of teammate counts for the same lot, and an export integration that pushes consolidated results to Google Sheets via the Sheets API. An in-app admin panel allows user management without direct database access.

  • C++

    rsLookup

    C++ / Bioinformatics

    A C++ command-line tool developed during undergraduate research at the University of Michigan to index and query the snp150Common genomic dataset by rsID or by chromosome, position, and allele sequence. To achieve efficient lookup over a multi-gigabyte dataset, the tool uses a custom disk-based two-way hash table backed by the diskhash library, with one-to-many mappings encoded through chained file pointer values as hash entries. The tool handles input normalization for complementary DNA strand representations and includes a Bash timing harness for performance validation. Presented at the University of Michigan UROP Spring Symposium, earning a Blue Ribbon Certificate.

  • C / Linux

    speechd-up (fork)

    C / Linux / Accessibility

    A fork of speechd-up, a discontinued C-based Unix daemon that bridges the speakup kernel module with the Speech Dispatcher userspace speech service on Linux. The original codebase had not been maintained since the mid-2000s and no longer built cleanly against modern GCC or current Speech Dispatcher headers. This fork restores buildability and adds synchronization logic that reads Speech Dispatcher's configured defaults for rate, pitch, and volume at startup and translates them into the corresponding speakup kernel parameters via the sysfs interface, with EINTR-safe I/O and a scale-translation function mapping between the two systems' parameter ranges. Also fixes three inherited bugs, including a sizeof pointer error in the index marker callback and a GCC 10+ linkage error caused by a global definition in a header.

  • Bash / Shell

    csvbwmon

    Bash / Shell / OpenWRT

    A bandwidth monitoring tool for OpenWRT-based routers that tracks per-device Internet usage by MAC address using iptables packet counters. Forked from an existing script and extended with monthly cycle archiving, arbitrary-precision integer arithmetic implemented in pure shell to handle exact byte counts without overflow, CSV export with spreadsheet formulas for per-device and total summaries across a date range, a lightweight PHP web GUI for exporting and managing device name mappings, and improved ARP handling to retain records for devices that are no longer active. Supports both external drive and NVRAM installation modes.
